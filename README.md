# InsurAI

InsurAI is an insurance workflow platform built to manage policies, claims, documents, risk assessment, fraud detection, and AI-assisted operations in one system.

## Live Demo

- Deployed App: `https://insurai-fwc6.onrender.com/`

## Project Structure

- `insurai-react` - React frontend
- `insurai-backend-java` - Java Spring Boot backend
- `insurai-backend` - earlier backend work and supporting files
- `report_assets` - screenshots used in the project report

## Features

- Login and authentication
- Policy creation with AI recommendation and risk score
- Claim creation with AI confidence
- Document upload and classification
- Risk assessment dashboard
- Fraud detection with active and resolved investigations
- AI assistant integration

## Tech Stack

- Frontend: React, Vite, HTML, CSS, JavaScript
- Backend: Java, Spring Boot
- Database: H2
- Tools: Maven, Git, GitHub

## Run Locally

### Frontend

```powershell
cd insurai-react
cmd /c npm run dev
```

### Backend

```powershell
cd insurai-backend-java
mvn spring-boot:run
```
